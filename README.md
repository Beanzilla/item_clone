# Item Replicator

Replicate items with ease.

## What's in the box

* [Unique API](API.md)
* [Internal Assitant Functions](INTERNALS.md)
* Multi-player support
* Security
* ALL item support

## Quick-Start

1. Get this repo!
2. Drop this directory into your Minetest mods folder.
3. Add `load_mod_item_clone = true` to your worlds world.mt file.
4. Enjoy the mod.
